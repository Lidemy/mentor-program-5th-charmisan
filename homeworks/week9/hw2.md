## 資料庫欄位型態 VARCHAR 跟 TEXT 的差別是什麼
預設開給所儲存文字的空間不同


## Cookie 是什麼？在 HTTP 這一層要怎麼設定 Cookie，瀏覽器又是怎麼把 Cookie 帶去 Server 的？
讓 sever 和 cient 端互相瞭解對方資訊的儲存空間，在 requset 或 response 中使用 set cookie 加在 header 中
